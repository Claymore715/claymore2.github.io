source

如果是js文件可以直接在sources里修改，用Ctrl+s保存后，可以直接生效。

如果非js文件，是在页面里写的js，可以把要修改的js函数的代码复制到console里执行，这样就会替换掉原js里的函数。这里要注意，是要以函数为单元修改。

如原来js的有一个函数

function aa（）

{

return tag>1

}

你可以这样做

在console里执行如下：

function aa（）

{

return true;

}
